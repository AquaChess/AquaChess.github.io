Politica de Privacidad

Información que recogemos

Información que recogemos de discord:

  • Id del usuario. El Id de usuario es un identificador único que está vinculado a cada usuario de Discord.
  • Nombre de usuario de Discord: El nombre de usuario es el nombre visible que cada usuario de Discord posee.
  • Nombre de usuario de chess.com y lichess.org: AquaBot permite al usuario vincular sus cuentas de chess.com y lichess.org.
  • Descripción del usuario: AquaBot permite al usuario agregar una descripción al perfil del usuario.
  • Juegos de lichess.org: Son las partidas jugadas en lichess.org que has enviado en un canal de texto o que ha sido usando en determinados comandos.
  • Torneos de lichess.org: Aqua almacena temporalmente links de torneos de lichess.org que han sido proporcionados por los usuarios.
  • Datos de uso: Se almacena el total de veces que un comando ha sido usado, se almacenan los datos en conjunto y no por usuario.
  • Partidas jugadas: Se almacenan las partidas jugadas por los usuarios usando a AquaBot.
  • Id de servidores: Cada servidor de Discord tiene asociado un identificador único.
  • Id de Roles de servidores: Se almacenan los identificadores de los roles de ciertos servidores al usar determinados comandos.

Información que recogemos de otras fuentes:

  • Puntuaciones elo: Aqua almacena las puntuaciones Elo de las categorías bullet, blitz y rapid de las cuentas vinculadas de chess.com y lichess.org.
  • Votos de Top.gg: Se almacenan el total de votos que ha realizado un usuario y la última vez que un usuario a votado por AquaBot.

Como usamos tu información

La información recolectada se usará para el correcto funcionamiento de AquaBot y para mejorarlo:

  • Id de usuario: El Id único de cada usuario se almacena para poder gestionar los datos que recopilamos de cada usuario, además nos permitirá poner en contacto con un usuario cuando sea necesario.
  • Personalización de tu perfil: AquaBot le permite al usuario crear un perfil personalizable, nosotros almacenamos usamos cierta información almacenada para mostrar ese perfil. La información que se muestra incluye una pequeña descripción y las cuentas enlazadas de chess.com y lichess.org con sus respectivo Elo de las categorías bullet, blitz y rapid. El usuario en cualquier momento puede modificar su descripción y desenlazar sus cuentas de chess.com y lichess.org. Existe una funcionalidad en AquaBot que permite al usuario ocultar su perfil haciéndolo privado.
  • Rankings: AquaBot usa los nombres de usuario para mostrarlos en el ranking. Existen 5 categorías de ranking que son ranking por puntucación elo, por número de votos, por la racha diaria, por el número de rompecabezas resuelto y por el número de monedas que el usuario posee. Los usuarios que tengan el perfil privado no serán mostrados en el ranking.
  • Juegos de lichess.org: AquaBot permite configurar los servidores para que cuando un usuario envíe una partida AquaBot responda enviando el gif de esa partida, además existen ciertos comandos que necesitan almacenar temporalmente la partida para su correcto funcionamiento.
  • Torneos de lichess.org: Existe un comando que permite al usuario agregar un torneo a la lista pública de torneos de AquaBot la cual puede ser vista por otro comando. Un torneo se almacenará hasta que esté empiece.
  • Partidas almacenadas: AquaBot permite a los usuarios jugar ajedrez dentro de discord por medio de ciertos comandos. Estás partidas serán almacenadas para que los usuarios puedan verlas cuando deseen.
  • Datos de servidores: AquaBot permite configurar los servidores, el id de los servidores se usan para mostrar mensajes automáticos y los id de los roles se usan para asignar roles de forma automática.
  • Puntuaciones elo de lichess.org y chess.com: se almacenan para mostrar un ranking de puntuaciones elo por servidor.
  • Tog.gg: Se detecta cada vez que alguien vota por AquaBot en top.gg, esto se hace para otorgar una recompensa en monedas para usarlas en la tienda de AquaBot, además se almacena el total de veces que un usuario ha votado para usarlo en el ranking de votos.

Como compartimos tu información

La información que añades se mostrará solamente a los miembros de Discord que accedan a ella desde algún comando.


Conservación de datos

La información será conservada durante el tiempo que sea necesaria para los fines para los cuales fueron recogidos.
